Projects on Console

Visual Studio empowers you to complete the entire development cycle in one place. For example, you can edit, debug, test, version control, and deploy to the cloud. With the diversity of features and languages in Visual Studio, you can grow from writing your first piece of code to developing in multiple project types.

How to set up Visual Studio:

  1. Make sure your computer is ready for Visual Studio.
  2. Download Visual Studio.
  3. Install the Visual Studio Installer.

In computer technology, a console can mean different things depending on the context. It usually means a combination of a display monitor and an input device, usually a keyboard and mouse pair, which allows a user to input commands and receive visual output from a computer or computer system.

To Open the console. You can do it in 2 different ways.

  • Press Control, Shift and I
  • Right Click and Click on Inspect

To print something in console screen, You need a .js file. In the Javascript, you can type the following: console.log("Anyting you want here")
